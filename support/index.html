Why we need donations

  • Server hosting: High-performance hardware and SSD storage keep lag low and player caps high.

  • Community events: Prizes, event infrastructure, and moderation tools help us run tournaments and seasonal events.
  • Fair access: Donations help keep the server free-to-play while offering optional donor perks that don't impact gameplay balance.
